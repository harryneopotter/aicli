### Getting Started

1. **Register or Login**
    - Access [Z.AI Open Platform](https://z.ai/model-api), Register or Login.

2. **Subscribe to GLM Coding Plan**
    - After logging in, navigate to the [GLM Coding Plan](https://z.ai/subscribe?utm_source=zai&utm_medium=link&utm_term=quickstart&utm_campaign=Platform_Ops&_channel_track_key=DRUfXN42) and choose your preferred subscription plan.

3. **Obtain API Key**
    - After subscribing, go to your account dashboard and click [API Keys](https://z.ai/manage-apikey/apikey-list) to generate a new API Key.
    - **Warning:** Safeguard your API Key by keeping it confidential. Avoid hard-coding it. Store in environment variables or configuration files.

4. **Select Coding Tool**
    - GLM Coding Plan supports multiple mainstream coding tools. Choose as per your preference:
        - [Claude Code]
        - [Roo Code]
        - [Kilo Code]
        - [Cline]
        - [OpenCode]
        - [Crush]
        - [Goose]
        - [Other Tools](https://docs.z.ai/devpack/tool/others)

5. **Configuring Coding Tools**
    - Example: _Using Claude Code_
        - **API Provider:** OpenAI Compatible
        - **Base URL:** `https://api.z.ai/api/paas/v4`
        - **API Key:** Enter your Z.AI API Key
        - **Model:** `GLM-4.6`
        - Refer to each tool's documentation for specific configuration steps.

6. **Start Coding**
    - Once configured, you can start coding with GLM-4.6!
        - Natural Language Programming
        - Code Debugging
        - Code Optimization

    **Example:**  
    ```
    # Using Natural Language Commands in Claude Code  
    Please create a React component containing a user login form
    ```
    GLM-4.6 will:
    - Analyze requirements and formulate a plan
    - Generate the complete React component code
    - Include validation and styling
    - Ensure the code runs directly
